The Queue: Deli Counter

html, css, javascript

This website was a capstone project combining HTML, CSS, and Javascript. The purpose of this "Deli Counter" is to accept customer orders, store them in a table that serves as an order queue. As orders are placed, the table displays customers in the order they were placed. After a customer has been served, a message is displayed and the remaining customers in the table move up in the queue. If the queue is empty, a message is display that no customers are waiting in line. Try out my Deli Counter for yourself!
